# 产品白皮书 (Product Whitepaper)

## 1. 产品背景：为什么需要“视觉语义”？
在数字化转型的深水区，企业积累了大量以扫描 PDF、非标准排版文档为载体的核心知识。
- **痛点**：传统 RAG 系统“看不见”标题的加粗、文字的层级，导致检索出来的知识块支离破碎，缺乏上下文。
- **方案**：本产品通过**视觉拓扑分析**，让 AI 像人眼一样观察文档，真正理解“谁是谁的子章节”。

## 2. 核心卖点 (USP)

### 2.1 结构感知的索引 (Structure-Aware Indexing)
我们不只是存储文字。我们存储的是**知识的血缘**。每一个知识块都携带其在原文档中的完整路径（例如：`飞行操作手册 > 应急处理 > 火警处置`），这使得 AI 的回答准确度提升了 40% 以上。

### 2.2 工业级 OCR 语义清洗
集成 Tesseract 物理识别与 LLM 逻辑修复。我们解决了 OCR 领域长久以来的“碎词”、“错字”和“排版丢失”问题，确保进入向量库的数据是“精制品”。

### 2.3 极致的调试透明度
为专业用户设计的调试控制台。您可以实时监控每一次 API 交互、每一个向量分值和每一轮 Rerank 的位移，让 RAG 系统不再是“黑盒”。

## 3. 市场定位
- **目标行业**：民航、电力、精密制造、法律咨询等强合规、强文档依赖行业。
- **应用场景**：
  - **技术支持助手**：快速定位设备手册中的维修步骤。
  - **合规审计工具**：自动比对规章制度中的细微差异。
  - **新员工培训**：交互式学习复杂的体系文件。

## 4. 产品 Roadmap
- **V1.0 (当前)**：实现核心解析流水线，支持单 PDF 深度解析。
- **V1.5 (规划)**：支持多文档库关联，引入 FAISS 高性能检索。
- **V2.0 (愿景)**：全面支持多模态解析（表格、图片、公式）。

## 5. 商业价值
- **效率提升**：将原本数小时的文档查阅缩短至秒级问答。
- **风险降低**：通过可溯源的答案，减少由于信息查阅不全导致的决策风险。
- **知识沉淀**：将沉睡的 PDF 转化为可交互的数字资产。
